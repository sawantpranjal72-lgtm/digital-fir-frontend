About Digital FIR System

Digital FIR System is an online government initiative designed to simplify the First Information Report registration process.

It allows citizens to file FIRs anytime, upload evidence securely, and track complaint status without visiting police stations.

The system ensures transparency, faster processing, and role-based access for Citizens, Police Officers, and Administrators.

User Roles

Digital FIR System provides role-based access for secure and efficient handling of cases.

👤
Citizen

Register complaints online, upload evidence, and track FIR status in real-time.

👮
Police Officer

Review FIRs, verify evidence, update case status, and take necessary actions.

🛠️
Administrator

Manage users, monitor system activities, and ensure smooth operations.

How Digital FIR System Works

1️⃣ Register / Login

Citizen logs in securely using email & password.

2️⃣ Create FIR

Fill FIR details and upload evidence online.

3️⃣ Police Review

Police verifies and processes FIR.

4️⃣ Track Status

Citizen tracks FIR status from dashboard.
